Creation and the use of the account / ENGLISH

●      The account is created by the user to use the services provided by the application.

●      A tree planter account is created once the user creates the account by signing up with an email and phone number.

●      After creating the account the user may access the different services provided by the application.

●      The company may collect personal information such as the Email address and the phone number.

●      No collected data will be shared with any third party.

●      Users are responsible for managing their IDs and passwords and shall be solely responsible for any changes or damages caused by intentional or unintentional acts. ●      Users shall only create one account.

Erstellung und Nutzung des Kontos/ GERMAN

●      Das Konto wird vom Benutzer erstellt, um die von der Anwendung bereitgestellten Dienste zu nutzen.

●      Ein “Plantree” konto wird erstellt, sobald der Benutzer das Konto erstellt, indem er sich mit einer E-Mail-Adresse und einer Telefonnummer anmeldet.

●      Nach der Erstellung des Kontos kann der Benutzer auf die verschiedenen Dienste zugreifen, die von der Anwendung bereitgestellt werden.

●      Das Unternehmen kann personenbezogene Daten wie die E-Mail-Adresse und die Telefonnummer erfassen.

●      Es werden keine erhobenen Daten weitergegeben.

●      Die Benutzer sind für die Verwaltung ihrer IDs und Passwörter verantwortlich und tragen die alleinige Verantwortung für alle Änderungen oder Schäden, die durch vorsätzliche oder unabsichtliche Handlungen verursacht werden. ●     Benutzer dürfen nur ein Konto erstellen.

Rules and obligations of the user/ English

●      Any violation of the rules set, will not be tolerated.

Users shall refrain from creating content like below:

-       a) Any content that causes fear and anxiety.

-       b) Any disrespectful remark to any group.

-       c) Any content that is not relevant to the purpose of the application.

-       d) Content whose purpose is aiding any crime.

-       e) Content that is contrary to public morals like swearing, any slurs, and violent content.

Users responsible for any of the content like in a),b),c),d),e) will get the alert the first time. If the behavior is repeated again the user's account will get deleted and removed.

Regeln und Pflichten des Benutzers / GERMAN:

●      Jeder Verstoß gegen die festgelegten Regeln wird nicht toleriert.

●      Benutzer dürfen keine Inhalte wie die folgenden erstellen:

a)  Jegliche Inhalte, die Angst und Unruhe hervorrufen.

b)  Jede respektlose Bemerkung gegenüber einer Gruppe.

c)  Jegliche Inhalte, die für den Zweck der Anwendung nicht relevant sind.

d)  Inhalte, deren Zweck darin besteht, Straftaten zu unterstützen.

The application is obligated to offer the following services:

-       Showing the user where to buy the trees.

-       Showing the user where to plant the trees.

-       Updating the user about the climate crisis.

The Admin of the application is able to delete, edit, and add any information at any time.

Account and password / ENGLISH

The account shall be reactivated if the password is not available for any kind of reason.

The user can delete their account and create a new account after.

The user is able t change the password at any time.

When the password is forgotten a message will be sent to the number entered.

Konto und passwort / GERMAN

Das Konto wird reaktiviert, wenn das Passwort aus irgendwelchen Gründen nicht verfügbar ist.

Der Benutzer kann sein Konto löschen und danach ein neues Konto erstellen.

Der Benutzer kann das Passwort jederzeit ändern.

Wenn das Passwort vergessen wird, wird eine Nachricht an die eingegebene Nummer gesendet.
